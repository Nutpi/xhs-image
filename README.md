# 📕 小红书图片生成器

> 输入 Markdown 内容，一键生成小红书风格的 3:4 图片切片，支持多套封面和内容模板。

## ✨ 功能特性

- **Markdown 输入** — 支持标准 Markdown 语法，保留粗体、斜体、代码等格式
- **富文本粘贴** — 从飞书、Notion、Word、网页直接粘贴，自动转为 Markdown
- **3:4 比例图片** — 生成 1080×1440px 高清图片，完美适配小红书
- **智能分页** — 自动根据内容长度分页，防止标题孤立
- **8 套封面模板** — 简约白 / 渐变醒目 / 暗黑高级 / 卡片杂志 / Claude 暖橙 / 金融时报 / 高迪·彩虹 / 书法雅韵
- **8 套内容模板** — 清新 / 暖纸 / 暗黑 / Claude 橙 / 金融时报 / 晚点红 / 高迪·彩色 / 书法
- **自适应标题字号** — 根据标题字数自动计算最佳字号，短标题超大字，长标题自动缩小
- **批量下载** — 支持单张下载和一键批量下载所有图片
- **丰富字体** — 加载 Google Fonts 中文字体（Noto Serif SC、Noto Sans SC、ZCOOL XiaoWei 等）

## 🚀 快速开始

### 方式一：本地运行

```bash
# 克隆项目
git clone https://github.com/Nutpi/xhs-image.git
cd xhs-image

# 启动本地服务
python3 -m http.server 8080

# 浏览器打开
open http://localhost:8080
```

### 方式二：直接打开

直接用浏览器打开 `index.html` 即可使用（部分功能可能需要 HTTP 服务器环境）。

## 📖 使用方法

1. **输入内容** — 在左侧编辑器输入 Markdown，或从其他应用直接粘贴富文本
2. **选择模板** — 在右侧选择喜欢的封面模板和内容模板
3. **生成图片** — 点击「🚀 生成图片」按钮
4. **预览下载** — 预览生成的图片，点击单张下载或一键下载全部

## 🛠 技术栈

- **Vue 3** — 前端框架
- **markdown-it** — Markdown 解析
- **html2canvas** — HTML 转图片
- **Turndown** — 富文本转 Markdown
- **Google Fonts** — 中文字体加载
